## 三、投毒比例消融实验

固定位置 `bottom_right`，投毒比例分别取 0.05、0.10、0.15，其余参数与主实验一致（10 epoch、目标类别 0、随机种子 2026）。

```bash
cd source_code
python -m experiments.run_poison_rate_ablation --position bottom_right --rates 0.05 0.10 0.15
```

| Poison Rate | Clean Accuracy | ASR | Training Time (s) |
|---:|---:|---:|---:|
| 0.05 | 0.7736 | 0.9443 | 498.32 |
| 0.10 | 0.7515 | 0.9688 | 617.60 |
| 0.15 | 0.7612 | 0.9703 | 559.49 |

随投毒比例升高，ASR 从 0.9443 增至 0.9703；Clean Accuracy 在 0.75～0.77 之间波动，未出现明显崩溃。0.10 与 0.15 的 ASR 接近，说明该位置在此范围内后门绑定已较充分。报告 4.6 节中 0.10 组与主实验 `bottom_right` 结果一致，可复现性良好。
