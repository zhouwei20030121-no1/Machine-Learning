# 图表说明

## 1. ASR 热力图（asr_heatmap.png）

### 图表用途

展示 3×3 不同触发器空间位置下的攻击成功率（Attack Success Rate, ASR）分布情况，用于分析触发器位置对后门攻击效果的影响。

### 实验结果分析

从热力图可以看出，所有位置的 ASR 均超过 0.92，说明 BadNets 后门攻击在各个位置均能够成功植入后门。

其中：

- middle_right 位置 ASR 最高，为 0.9820；
- center 位置 ASR 为 0.9817；
- middle_left 位置 ASR 为 0.9747；

上述三个位置均位于图像中部区域。

相比之下：

- top_left 位置 ASR 为 0.9298；
- top_right 位置 ASR 为 0.9310；

属于整体较低水平。

实验结果表明，触发器位于图像中部区域时更容易被卷积神经网络学习，从而获得更高的攻击成功率；位于图像边缘区域时，攻击效果略有下降。

## 2. Clean Accuracy 与 ASR 对比图（acc_asr_bar.png）

### 图表用途

比较不同触发器位置下模型正常分类准确率（Clean Accuracy）与攻击成功率（ASR），分析后门攻击对模型正常性能的影响。

### 实验结果分析

从柱状图可以看出：

- Clean Accuracy 整体保持在 0.71～0.77 之间；
- ASR 保持在 0.93～0.98 之间。

说明在固定投毒比例（10%）条件下，BadNets 攻击能够在保持较高正常分类性能的同时实现较高攻击成功率。

不同位置之间的 Clean Accuracy 波动较小，最大差异约为 5%，表明触发器位置对模型正常分类能力影响有限。

同时，中部区域位置（center、middle_left、middle_right）具有更高的 ASR，说明这些位置能够在不明显降低模型性能的情况下实现更强的攻击效果。

## 3. 检测难度热力图（detection_heatmap.png）

### 图表用途

展示不同触发器位置对应的 Detection Score，用于分析触发器位置对后门检测效果的影响。实验采用 Neural Cleanse 方法对训练后的后门模型进行检测，并以 Detection Score 作为检测能力指标。

### 实验结果分析

从热力图可以看出，不同位置对应的 Detection Score 均处于较高水平（0.929～0.981），说明 Neural Cleanse 能够较为有效地恢复和识别本实验构造的 BadNets 后门触发器。

其中：

-  center 位置 Detection Score 最高，为 0.9806； 
-  middle_right 位置 Detection Score 为 0.9806； 
-  middle_left 位置 Detection Score 为 0.9737； 

上述位置均位于图像中部区域。

相比之下：

-  top_left 位置 Detection Score 为 0.9290； 
-  top_right 位置 Detection Score 为 0.9303； 

属于整体较低水平。

实验结果表明，位于图像中部区域的触发器不仅具有较高的攻击成功率，同时更容易被检测算法恢复和识别；而位于图像边缘区域的触发器检测得分略低，具有一定的隐蔽优势。

总体来看，触发器位置会影响后门检测效果，中部区域检测难度相对较低，而边缘区域检测难度相对较高。

## 4. ASR 与检测难度权衡图（asr_detection_tradeoff.png）

### **图表用途**

联合分析攻击成功率（ASR）与防御检测得分（Detection Score）之间的空间分布关系，用以定量评估不同触发器植入位置在“攻击破坏力”与“防御隐蔽性”之间的博弈与权衡情况。

### **实验结果分析**

从散点图可以看出，ASR 与 Detection Score 呈现明显的正相关趋势。

具体而言：

-  center、middle_left、middle_right 等中部位置同时具有较高的 ASR 和较高的 Detection Score； 
-  top_left、top_right 等边缘位置同时表现出较低的 ASR 和较低的 Detection Score。 

这表明：

1.  当触发器更容易被模型学习时，通常也更容易被检测算法发现； 
2.  当触发器位于图像边缘区域时，其攻击效果略有下降，但同时检测难度有所增加； 
3.  攻击效果与隐蔽性之间存在一定的权衡关系。 

从综合性能角度来看：

-  center 和 middle_right 位置具有最强的攻击能力； 
-  top_left 和 top_right 位置具有相对更好的隐蔽性； 
-  bottom_center 位置在攻击成功率与检测难度之间表现较为均衡。 

因此，在实际后门攻击场景中，需要根据攻击目标选择不同的触发器位置，以平衡攻击成功率与隐蔽性之间的关系。

## 5. 投毒比例消融图（poison_rate_ablation.png）

### 图表用途

分析不同投毒比例（Poison Rate）对模型正常分类准确率（Clean Accuracy）和攻击成功率（Attack Success Rate，ASR）的影响，从而研究后门攻击强度与模型正常性能之间的关系。

### 实验结果分析

如图所示，在固定触发器位置（**bottom_right**）下，分别设置投毒比例为 **5%、10% 和15%**，得到模型在不同投毒比例下的 Clean Accuracy 与 ASR 变化情况。

从实验结果可以看出：

-  当投毒比例由 **5%** 增加至 **10%** 时，ASR 由 **0.9443** 提升至 **0.9688**； 
-  当投毒比例继续增加至 **15%** 时，ASR 略微提升至 **0.9703**。 

说明随着投毒样本数量增加，模型能够更加充分地学习触发器与目标类别之间的关联，从而提高攻击成功率。但当投毒比例达到一定程度后，ASR 的提升趋于平缓，说明继续增加投毒比例带来的收益有限。

与此同时，模型的 Clean Accuracy 整体保持在 **0.75～0.77** 之间：

-  **5%** 投毒比例下最高，为 **0.7736**； 
-  **10%** 时下降至 **0.7515**； 
-  **15%** 时回升至 **0.7612**。 

整体波动幅度不足 **3%**，说明在本实验设置下，适度增加投毒比例不会明显降低模型对正常样本的分类性能。

综合来看，投毒比例会直接影响后门攻击效果。当投毒比例较低时，攻击成功率相对有限；随着投毒比例增加，ASR 明显提高，而模型正常分类性能基本保持稳定。因此，在本实验中，**10% 左右的投毒比例能够在保证较高攻击成功率的同时维持较好的正常分类性能，是攻击效果与模型性能之间较为合理的折中选择。**

## 当前结论

基于已完成的全部实验结果，可以得到如下结论：

1.  所有触发器位置均能够成功实现 BadNets 后门攻击，攻击成功率（ASR）均高于 **92%**； 
2.  图像中部区域（**center、middle_left、middle_right**）具有更高的攻击成功率，说明触发器位置会影响后门特征的学习效果； 
3.  不同触发器位置对模型正常分类性能影响较小，Clean Accuracy 始终保持在较高水平； 
4.  Neural Cleanse 检测结果表明，中部位置具有更高的 Detection Score，更容易被检测算法识别，而边缘位置具有一定的隐蔽优势； 
5.  攻击成功率与检测难度之间存在一定的权衡关系，中部位置攻击效果更强但检测难度较低，边缘位置攻击效果略弱但隐蔽性更好； 
6.  投毒比例增加能够提高攻击成功率，但当投毒比例达到约 **10%** 后，ASR 提升逐渐趋于饱和，而模型正常分类准确率变化较小，说明适中的投毒比例能够较好地兼顾攻击效果与模型性能。
