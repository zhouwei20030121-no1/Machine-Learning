# ASR 与 Detection Score 权衡分析

## 1. 分析目标

ASR 表示攻击成功率，数值越高，说明触发器越容易让模型输出攻击者指定的目标类别。Detection Score 表示检测显著性，数值越高，说明触发器激活后门的特征越明显，也越容易被当前检测方法发现。

因此，一个触发器位置是否“好”，不能只看 ASR。理想情况下，攻击者希望 ASR 高、Detection Score 低；但如果 ASR 和 Detection Score 都高，就说明攻击有效，但隐蔽性较差。

## 3. 权衡关系

从整体结果看，9 个位置都表现为高 ASR 和高 Detection Score。这说明 3x3 白色触发器在所有位置上都能有效诱导模型输出目标类别，但这种后门激活特征也比较明显，容易被当前检测方法捕捉。

center 和 middle_right 的 ASR 分别为 0.9817 和 0.9820，是攻击效果最强的两个位置；它们的 Detection Score 也分别达到 0.980641 和 0.980572，说明攻击越强的位置往往检测显著性也越高。

top_left 和 top_right 的 ASR 分别为 0.9298 和 0.9310，是 9 个位置中较低的两个位置；它们的 Detection Score 也最低，分别为 0.929033 和 0.930278。因此从相对角度看，这两个位置更隐蔽，但攻击强度也弱一些。

## 4. 结论

本实验没有出现“高 ASR + 低 Detection Score”的理想隐蔽攻击位置。所有位置都可以成功触发后门，但也都比较容易被检测出来。

如果优先考虑攻击成功率，center 和 middle_right 更优；如果优先考虑相对隐蔽性，top_left 和 top_right 更优。但由于 top_left 和 top_right 的 Detection Score 仍然超过 0.92，它们只能说是相对更隐蔽，不能认为是真正难检测。

## 5. 可直接放入报告的文字

将 ASR 与 Detection Score 联合分析可以发现，9 个触发器位置整体呈现出“攻击成功率高、检测分数也高”的特点。center 和 middle_right 的 ASR 最高，同时 Detection Score 也最高，说明它们攻击效果最强，但后门激活特征也最明显。top_left 和 top_right 的 ASR 与 Detection Score 相对较低，说明它们在本实验中相对更隐蔽，但由于 Detection Score 仍然高于 0.92，因此并不能认为是真正难以检测。

总体来看，本实验中的 3x3 白色触发器具有较强攻击效果，但隐蔽性一般。触发器位置会影响攻击成功率和检测分数，但在当前模型和数据集设置下，不同位置都没有表现出强隐蔽性。
